Projects




Jumper


Jumper is a software that allows you to save your credit cards, passwords and telephone numbers in a SQL database on your pc. This means that your data will only be viewable through the software. It's required a login and multiple accounts can be created. It has a simple GUI, made with Python and CustomTkinter, so everyone can use it without any problem. Also for security purpose the data contained in the database is encrypted with asymmetric keys.

Please read very carefully the instructions.

Just Run!


"Just Run!" it's the first video game I've ever made. It consists of a simple but fun car game set in an original game map. There are no challenges or rankings as they generally tend to annoy the player, so there is only a freeplay mode. You can run very fast or do some breathtaking drift trying not to crash into a light pole. Also in the game menu you can choose which car to use. The video game was created using Unity and C# and is available online on the official Unity website.
